## Getting Started

**1- Clone this repository:**
```bash
https://github.com/luizfranzon/twitter-video-downloader && twitter-video-downloader/App
```
**2- Install dependencies:**
```bash
npm install
```

**3- Run dev server:**
```bash
npm run dev
```
